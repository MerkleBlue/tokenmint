DEPLOYMENT

Since the app is not production ready, it cannot be found on the Chrome Web Store.
In order to deploy the app, apply the following steps:

  1. Download the zipped extension
  2. Unpack the zipped extension anywhere on your local machine
  3. Click on Customize and control Google Chrome button
  4. Go to More tools->Extensions
  5. Click on Load unpacked button
  6. Select chrome folder in the unpacked structure and click OK
  7. Refresh tabs
  8. Exercise your right of free speach
